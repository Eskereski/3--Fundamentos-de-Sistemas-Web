# Petshop Web - Fase 2

## Descrição
Evolução do sistema web estático para um petshop, integrando design responsivo e interatividade. 

## Funcionalidades e Ajustes Realizados na Fase 2
- **Integração do Bootstrap 5:** Utilizado para a criação do layout responsivo e adição do componente de Carrossel para destacar serviços.
- **Formulário de Agendamento:** Criação de um formulário completo (dados do cliente e do pet) com inputs variados (text, email, tel, radio, checkbox, date, time).
- **Acessibilidade:** Inclusão de atributos `aria-label` no formulário e manutenção dos atributos `alt` nas imagens.
- **Interatividade (JavaScript):** - Função temporal (`setTimeout`) que exibe um Pop-up promocional 5 segundos após o carregamento da página.
  - Prevenção do envio padrão do formulário (`preventDefault`) e exibição de alerta customizado de sucesso (Modal Bootstrap).
- **Estilização (CSS):** Criação de uma paleta de cores própria focada no tema do petshop e ajustes de espaçamento.

## Tecnologias
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
